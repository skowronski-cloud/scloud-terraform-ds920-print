LABEL PRINTSERVER

  • Windows
    • General
      1. To solve Microsoft Windows Network: you cannot access this shared folder because your organisation's security policies block unauthenticated guest access. click on AllowInsecureGuestAuth.reg
      2. Connect to \\${ip} and map your printers
    • DYMO Specific
      1. Open \\${ip}\dymo and install ".exe" file
  • macOS
    • General
      1. Just open System Preferences.app, find Printers & Scanners, click Add button and add those ending with "@ print".
    • DYMO Specific
      1. brew install dymo-label
      2. or install ".pkg" file from smb://${ip}/dymo (log in as Guest)
  • Troubleshooting
    • Open http://${ip}:631/printers/
